LET OP

Open Transformatie NL is een community project

Open Transformatie NL is een community project om een overzicht te maken van de verschillende initiatieven die gezamenlijk de digitale transformatie van de overheid mogelijk maken op een open en eerlijke manier. Dit project is niet door de overheid geïnitieerd.
  • NL Design System

    Gezamenlijk design system voor Nederlandse overheden.
  • Code for NL Collecties

    Use Cases van open applicaties in de overheid.
  • Developer Overheid

    Eén centrale plek voor de developer die voor of met de overheid ontwikkelt.
  • Standard for Public Code

    The Standard for Public Code gives cities a model for building their own open source solutions to enable successful future reuse by other cities.
  • Common Ground

    Hervorming van de gemeentelijke informatievoorziening, door op een andere manier om te gaan met gegevens.
  • Data Overheid

    Dataregister van de Nederlandse Overheid.
  • Object Detection Toolkit

    Open AI toolkit voor slimme applicaties.
  • Innovatiebudget Digitale Overheid

    Ondersteuning voor innovatie van digitale dienstverlening en oplossen van vraagstukken.
  • Code for NL

    (Slack)community van developers en designers die samenwerken aan de digitale overheid.
  • Piano Triennale

    Het Italiaanse voorbeeld voor dit initiatief.
